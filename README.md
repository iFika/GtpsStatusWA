# GtpsStatusWA
GTPS Status For Whatsapp Web.

# Usage
> 1. Install Node JS Stable.
> 2. Edit Setting.json. And Edited Destination Folder And Name Computer
> 3. Running a Bot.
> 4. Scan QR Code On CMD For Login To Web Whatsapp.
> 5. And Enjoyed.
